# 12daysofjelly

Very basic express server running on nodeJS 
server/index.js (currently) sets up the 

## Setup:

```
cd server 
npm install
```

To run:

```
cd server
npm start
 ```
(starts express server)

or 

```
cd server
npm run dev
```
(starts in dev mode, edits to index.js update automatically)
